Macro Definition Documentation

◆ GIF_FREE

#define GIF_FREE   free

Макрос для освобождения памяти.

Этот макрос используется только в функции GifEnd для освобождения буфера размером изображения, выделенного функцией GifBegin.

◆ GIF_MALLOC

#define GIF_MALLOC   malloc

Макрос для выделения памяти.

Этот макрос используется только в функции GifBegin для выделения буфера размером изображения, который используется для выявления измененных пикселей при дельта-кодировании.

◆ GIF_TEMP_FREE

#define GIF_TEMP_FREE   free

Макрос для освобождения временной памяти.

Если вы хотите использовать собственный аллокатор памяти, переопределите этот макрос перед включением этого заголовочного файла. По умолчанию используется стандартная функция free.
